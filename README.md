<h1 align="center">Telar Mongo in TypeScript</h1>

<div align="center">
 Telar Mongo is a wrapper for MongoDB JS sdk.
</div>

## Installation

Telar Mongo is available on npm. You can install it using:

```sh
npm install @telar/mongo
#or
yarn add @telar/mongo
```
